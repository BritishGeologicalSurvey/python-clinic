# Advent of Code Dojo

> Input files and other data for Advent of Code dojo.

+ [Advent of Code website](https://adventofcode.com/)

## Setup

The `pytest`, `ipython` and `ipdb` packages can be installed with:

```bash
pip install -r requirements.txt
```
